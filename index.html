An online Pomodoro Timer to boost your productivity

What is Pomofocus?

Pomofocus is a customizable pomodoro timer that works on desktop & mobile browser. The aim of this app is to help you focus on any task you are working on, such as study, writing, or coding. This app is inspired by Pomodoro Technique which is a time management method develoed by Francesco Cirillo.

What is Pomodore Technique?

The Pomodoro Technique is created by Francesco Cirillo for a more productive was to work and study. The technique uses a timer to break work inro intervals, tradutionally 25 minutes in lenght, seperated by short breaks. Each interval is know as a pomodoro from the Italien word for "tomato", after the tomota-shaped kitchen timer that Cirillo uased as a university studen. - Wikipedia

How to use the Pomodoro Timer?

  1. Add tasks to work today
  2. Set estimate pomodoros (1=25min of work) for each task
  3. Select a task to work on
  4. Start timer and focus o the task for 25 minutes
  5. Take a break for 5 minutes when the alarm ring
  6. Iterate 3.5 times until you finish the tasks

Features

  • Responsive design that works with desktop and mobile
  • Color tanstition to switch moods between work time and rest time
  • Audio notification at the end of a timer period
  • Customizable timer intervals to suit your preference
